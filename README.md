## 📂 Τι περιέχει το project

### 🌐 `index.html` — Το Διαδικτυακό Κουίζ

Μια **single-file web εφαρμογή** με Marvel θεματολογία.
- **15 ερωτήσεις** από μαθηματικά, λογική και Marvel ήρωες
- Κάθε λάθος απάντηση επαναλαμβάνει την ερώτηση — δεν προχωράμε χωρίς σωστή απάντηση!
- Comic-book σχέδιο με bold χρώματα και animation
- Ήχοι μέσω Web Audio API (χωρίς εξωτερικά αρχεία)
- Λειτουργεί σε κινητό, tablet και υπολογιστή

### 🖥️ `konstantinos.py` — Το Terminal Κουίζ

Ένα **Python πρόγραμμα** για Windows terminal.
- **10 ερωτήσεις** με χρωματιστές απαντήσεις (colorama)
- Ήχοι μέσω `winsound` (built-in Windows)
- Δραματικό φινάλε με έκπληξη!

---

## 🚀 Πώς να τρέξεις

### Το web κουίζ (`index.html`)
Απλώς κάνε **διπλό κλικ** στο `index.html` και ανοίγει στον browser σου!
Δεν χρειάζεται server ή εγκατάσταση.

### Το terminal κουίζ (`konstantinos.py`)

```bash
# Βεβαιώσου ότι έχεις Python εγκατεστημένο, έπειτα:
pip install colorama

# Και τρέξε:
python konstantinos.py
```

> **Σημείωση:** Το `winsound` είναι ήδη ενσωματωμένο στα Windows — δεν χρειάζεται εγκατάσταση.
> Αν το `colorama` δεν είναι εγκατεστημένο, το πρόγραμμα θα τρέξει κανονικά χωρίς χρώματα.
